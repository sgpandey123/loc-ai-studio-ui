# Build a Jira-Inspired Loan Origination System (LOS) Frontend

Create a modern enterprise-grade Loan Origination System (LOS) frontend application using:

* Frontend Framework: Thymeleaf-based UI templates
* Design Style: Jira-inspired workflow management platform
* Theme: Premium Banking UI with Red (#C8102E), White, and Light Gray color palette
* Layout: Responsive Web Application
* UI Style: Clean, enterprise, professional banking software
* Icons: Lucide Icons
* Components: Cards, Modals, Rich Text Editors, Task Panels, Activity Timelines
* Drag & Drop: Full Kanban functionality similar to Jira

Build a Loan Origination Platform where loan applications move through multiple workflow stages .

The experience should feel like:

* Jira
* Monday.com
* Azure DevOps Boards
* Salesforce Pipeline Management

combined with banking loan workflow requirements.

# Stage 2: Customer Onboarding

When application moves into Customer Onboarding:

System automatically creates onboarding tasks.

Default Tasks:

### Document Collection

* PAN
* Aadhaar
* Income Proof
* Bank Statements
* Address Proof
* Business Registration Documents

Each task contains:

* Status
* Assignee
* Due Date
* Attachments
* Comments

Task States:

* To Do
* In Progress
* Completed

All onboarding tasks must be completed before moving further.

Display progress bar:

8/8 Documents Collected

100% Complete

# Stage 3: Underwriting

When application enters Underwriting:

Automatically create credit tasks.

Default Tasks:

1. Hygiene Check
2. Field Visit
3. Financial Analysis
4. CRAN Generation
5. Terms & Conditions Preparation

### Rich Text Credit Assessment Editor

* Tables

* Image uploads
